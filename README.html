👁️ AI 기반 안구 질환 통합 진단 시스템

Jetson/Raspberry Pi 기반 엣지 AI 안구 진단 웹 애플리케이션
실시간 검출 · 질환 분류 · 충혈도 분석 · 히스토리 리포트

주요 기능

  • 실시간 카메라 스트리밍 및 눈 감지 상태 표시 (/video_feed, /detect_status)
  • 얼굴 단일 촬영/업로드 기반 진단 파이프라인 (YOLO가 좌/우 눈 자동 감지 및 크롭)
  • 업로드 이미지 분석 API (/analyze)
  • 사용자별 진단 히스토리 조회 (/api/history)
  • 설문(문진) 데이터 저장 (/api/survey)
  • 분석 결과 기반 AI 가이드(위험도/권장 진료/관리 수칙) 자동 생성
  • 리포트 PDF 생성 및 카카오톡 공유 (/api/report/share)
  • QR + PIN 기반 O2O 모바일 인증 연동 (/m, /m/dashboard)
  • 관리자 설정 페이지 및 설정 저장 API 제공

최근 업데이트 (2026-03 기준)

  • 리포트 화면: 전체 히스토리 목록 스크롤, 비교 슬롯 매핑 안정화, 크롭 이미지 표시 강화
  • capture/result/report 포함 주요 화면 반응형 개선 및 다크 테마 통합
  • database/backfill_guides.py 백필 스크립트 추가

기술 스택

Flask SQLite3 PyTorch torchvision Ultralytics YOLOv8 OpenCV NumPy Pillow python-dotenv requests reportlab qrcode.js HTML/CSS/JS

실행 방법

1) Python 환경

python3 -m venv .venv
source .venv/bin/activate

2) 패키지 설치

pip install flask opencv-python numpy pillow torch torchvision ultralytics python-dotenv requests reportlab

3) 환경변수 (권장)

export HASH_PEPPER='change-this-secret'
    export EYE_APP_SECRET_KEY='change-this-secret-too'
    export KAKAO_ACCESS_TOKEN='kakao-access-token'

4) 서버 실행

python eye_server.py
접속: http://0.0.0.0:5000 또는 장비 IP

핵심 라우트

페이지

  • GET /, /login, /admin/config, /capture, /result, /report, /survey, /m, /m/dashboard

API

  • GET /status, /video_feed, /video_frame, /detect_status
  • POST /analyze, /diagnose
  • GET /capture/state, POST /capture/reset
  • GET /api/history, POST /api/survey
  • POST /api/report/share, POST /api/generate_pin, GET /api/pin_status, POST /api/mobile_connected, POST /api/verify_pin
  • POST /api/admin/login, GET/POST /api/admin/config

데이터 저장

  • DB 파일: database/history.db
  • 이미지: web/static/captures/users/<user_id>/...
  • 테이블: diagnosis_history, survey_history
  • /analyze 저장 데이터에 guide 필드 포함

백필 스크립트

과거 히스토리 레코드에 guide 필드를 보강할 때 사용
# dry-run
python database/backfill_guides.py --db database/history.db --dry-run

# apply
python database/backfill_guides.py --db database/history.db
