# eye_project

👁️ 안구 진단 프로젝트 (Team Papyrus)
세은아! 우리 프로젝트 깃허브에 온 걸 환영해. VS Code 설치 없이 웹에서 바로 수정할 때 아래 가이드를 참고해 줘!

📂 폴더 및 파일 위치 안내
파일을 수정하거나 업로드할 때 꼭 정해진 폴더 안에 넣어줘!
HTML 파일들 (web/templates/ 폴더)
layout.html: 모든 페이지의 공통 뼈대 (헤더, 푸터)
index.html: 메인 화면
login.html: 전화번호 입력 화면
survey.html: 문진표 작성 화면
capture.html: [중요] 안구 촬영 화면 (가이드라인 포함)
result.html: 최종 결과 확인 화면
디자인 파일들 (web/static/ 폴더)
CSS: web/static/css/ 안에 style.css 파일을 수정해 줘.
이미지: 로고나 배경 이미지는 web/static/images/ 폴더에 올려줘.
JS: 버튼 클릭 이벤트 등은 web/static/js/ 폴더를 사용해.
🛠️ 웹에서 바로 수정하는 꿀팁 (2가지)
방법 1: 초간단 웹 에디터 (강력 추천!)
이 화면에서 키보드의 **마침표(.)**를 딱 한 번만 눌러봐.
그러면 브라우저 안에서 VS Code랑 똑같이 생긴 편집창이 열려! 여기서 파일을 고치고 왼쪽 메뉴에서 '저장(Commit)'하면 바로 반영돼.
방법 2: 개별 파일 수정
수정하고 싶은 파일을 클릭한 뒤, 오른쪽 상단의 **연필 아이콘(Edit this file)**을 눌러서 코드를 붙여넣고 저장하면 돼.
⚠️ 디자인 주의사항
해상도: 모든 화면은 640x360 해상도를 기준으로 작업해 줘.
촬영 가이드라인: capture.html 작업할 때 눈 위치를 잡는 박스는 반드시 아래 수치를 지켜줘야 AI가 제대로 인식해!
좌표: x=130,y=90
크기: 가로 380px, 세로 180px
